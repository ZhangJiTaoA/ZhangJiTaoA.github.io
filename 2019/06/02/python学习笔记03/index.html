一. 文件操作

1. 基本读写文件的方式

文件的读和写操作通过文件中的光标来定位，从而知道该从哪里写，从哪里读。首先介绍一下相应的方法
f.tell() # 返回光标当前位置，一个英文字符占一个，中文占三个(标准，是编码的关系)
f.seek(4) # 调整光标位置，这里表示到第四个字节的位置
f.truncate(10) # 截断，打开方式应该为’a’，表示从第十个字节之后的都删除，只保留前十个字节
